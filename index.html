Documentación de la API de Productos

Descripción General

Esta API permite gestionar productos, incluyendo la creación, actualización y eliminación de productos en una base de datos MongoDB. Utiliza el framework Flask para crear un servidor web que ofrece una interfaz para interactuar con los productos.

Endpoints

1. Obtener la lista de productos

Ruta: /

Método HTTP: GET

Descripción: Esta ruta muestra una lista de productos almacenados en la base de datos.

Ejemplo de respuesta JSON: 
[
    {
        "name": "Producto 1",
        "price": 10.99,
        "quantity": 50
    },
    {
        "name": "Producto 2",
        "price": 19.99,
        "quantity": 30
    }
]

2. Agregar un nuevo producto

Ruta: /products

Método HTTP: POST

Descripción: Permite agregar un nuevo producto a la base de datos.

Parámetros del cuerpo (JSON):

Ejemplo de solicitud JSON: 
{
    "name": "Nuevo Producto",
    "price": 15.99,
    "quantity": 20
}

3. Eliminar un producto

Ruta: /delete/<string:product_name>

Método HTTP: GET

Descripción: Permite eliminar un producto de la base de datos.

Parámetros de la URL:

Ejemplo de solicitud URL: 
/delete/Producto1

4. Editar un producto existente

Ruta: /edit/<string:product_name>

Método HTTP: POST

Descripción: Permite editar un producto existente en la base de datos.

Parámetros de la URL:

Ejemplo de solicitud URL: 
/edit/Producto1

Parámetros del cuerpo (JSON):

Ejemplo de solicitud JSON: 
{
    "name": "Producto Actualizado",
    "price": 17.99,
    "quantity": 25
}

Librerías Utilizadas

  • Flask: Flask es un framework web de Python utilizado para crear aplicaciones web.
  • pymongo: La biblioteca pymongo se utiliza para interactuar con la base de datos MongoDB.
  • certifi: Certifi es una biblioteca utilizada para la gestión de certificados de seguridad.
